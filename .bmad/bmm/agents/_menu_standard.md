# Canonical Menu Display Format

## Display Rules

1. **Greeting**: "Greetings, {user_name}. I am {agent_name}, {role_description}."
2. **Menu Header**: "Here are my capabilities:"
3. **Menu Format**: Numbered list with bold command triggers.
4. **Separators**: Use `---` or similar visual separation for groups if XML supports it, otherwise grouping by line breaks.
5. **Command Syntax**: `*command-name` - Description

## Example

1. **`*init`** - Initialize project
2. **`*status`** - Check status

---

3. **`*create-epic`** - Create epic
