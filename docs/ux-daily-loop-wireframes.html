Microcopy: All Caught Up

Primary Message
"You're all caught up"
Simple, confident. Not "Nothing to do!" (feels dismissive) or "Great job!" (feels patronizing).
Supporting Message
"Xentri handled 12 items since yesterday. Nothing needs your attention."
Shows value delivered. "Handled" implies competence. "Nothing needs your attention" = permission to close the app.
Alternative (No Activity)
"All quiet. Check back later or review your timeline."
For truly empty states (new user, no activity).

Undo Behavior Spec

Undo Mechanics

Toast Pattern
"[Action]. [Object]" + "Undo" link
Examples: "Sent. Reply to Sarah" / "Archived. 3 leads" / "Published. Homepage"
Undo Window
5 seconds for toast, unlimited in timeline
Toast auto-dismisses after 5s. User can always undo from timeline (if action is reversible).
Undo for Emails
"Retract" sends a correction email
"Oops, please disregard my last message. Here's the correct info:" — only if within 2 minutes.
Non-Reversible Actions
Marked clearly before execution
Payments, legal docs, permanent deletes show "This cannot be undone" warning.

Microcopy: Undo

Success + Undo
"Sent. Undo" / "Done. Undo" / "Archived. Undo"
Terse. Confident. Not "Successfully sent!" (too eager).
After Undo
"Undone. Back in your queue."
Confirms the undo worked and where to find it.
Undo Expired
"Can't undo — already delivered. Send a correction?"
Offer the next best action.

"/contact form" / "WhatsApp forward" / "Manual entry"
Confidence

Actor

"You" / "Autopilot" / "System" / "Team member name"
