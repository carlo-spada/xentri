# Research Documents

> **Purpose:** Track market, competitive, and user research for the Xentri platform.

---

## Overview

This directory contains research documents that inform product decisions. These are referenced by the PRD and should be updated periodically.

## Research Types

| Type | Document | Status |
|------|----------|--------|
| **Market Research** | [market-research.md](./market-research.md) | Pending |
| **Competitive Analysis** | [competitive-analysis.md](./competitive-analysis.md) | Pending |
| **User Research** | [user-research.md](./user-research.md) | Pending |

## Research Process

Use the BMAD research workflow to conduct structured research:

```bash
/bmad:bmm:workflows:research
```

## Document Format

Each research document should include:

1. **Metadata** — Research date, methodology, sources
2. **Executive Summary** — Key findings
3. **Detailed Findings** — Data and analysis
4. **Implications** — How findings affect product decisions
5. **Next Steps** — Recommended actions

## Update Cadence

- **Market Research:** Quarterly
- **Competitive Analysis:** Quarterly
- **User Research:** Ongoing (after each significant user interaction)
