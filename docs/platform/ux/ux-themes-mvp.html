Theme Comparison Notes

Modern Theme

  • Chronicle view by default (narrative-first)
  • Copilot appears with avatar, feels like colleague
  • Gradient accents create immersive feel
  • Comfortable spacing, breathing room
  • Full sidebar with labels
  • Friendly, conversational typography

Power Theme

  • Efficiency view by default (data-forward)
  • Copilot as inline notes, no avatar
  • Muted colors, high contrast
  • Compact spacing, more on screen
  • Icon-only sidebar saves space
  • Table-based data presentation

Light vs Dark

  • Same structure, inverted colors
  • Light modes use slightly deeper primaries
  • Contrast ratios maintained for accessibility
  • Border colors adjust for visibility

User Preference

  • Theme stored per user (not per org)
  • Default: Modern Dark
  • Toggle available in Settings
  • Brief-aware suggestions possible
